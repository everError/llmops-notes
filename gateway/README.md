# Gateway

여러 모델·프로바이더(로컬 vLLM, Ollama, 클라우드 API)를 **단일 OpenAI 호환 엔드포인트**로 묶는 라우팅 계층입니다.

애플리케이션은 게이트웨이만 바라보고, 모델 교체·폴백·비용 관리는 게이트웨이 설정으로 처리하는 것이 목표입니다.

## 다룰 도구

- **LiteLLM** — 사실상 표준 LLM 프록시. 100+ 프로바이더 지원, OpenAI 호환

## 앞으로 다룰 주제

- [ ] LiteLLM Proxy 셀프호스팅 (docker compose)
- [ ] 로컬 vLLM + Ollama를 하나의 엔드포인트로 통합
- [ ] 폴백 체인 — 로컬 모델 장애 시 클라우드 API로 폴백
- [ ] 레이트리밋, API 키별 예산·비용 추적
- [ ] 응답 캐싱
